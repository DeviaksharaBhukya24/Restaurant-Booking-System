TheRestaurant Booking System is designed to simplify and automate the process of
managing table reservations and customer queues in a restaurant. In this system, every
customer is provided with a token number upon booking. Depending on table availability, the
customer is either allotted a table immediately or placed on a waiting list. The system keeps
track of all tables, monitors their availability, and updates the waiting list dynamically. It also
includes a fixed menu with item prices, allowing customers to select items and quantities,
after which the total bill is automatically calculated. If a customer finishes dining, the system
removes them from thequeueand seats thenext waiting customer. VIP customers are given
special priority for seating. The entire process is efficiently implemented using queues with
linked lists, which ensures smooth and fair handling of customers. This project
demonstrates how data structures can be applied to solve real-life management problems
effectively
